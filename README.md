## ğŸ” CI/CD Pipeline

CI/CD is powered by GitHub Actions.

### Workflow: `.github/workflows/deploy.yml`

**Stages:**
1. Checkout code
2. Install dependencies
3. Build Docker images
4. (Optional) Run backend unit tests
5. SCP files to EC2
6. SSH into EC2 and redeploy via Docker Compose

**Secrets Required:**
- `SSH_PRIVATE_KEY`
- `EC2_HOST`
- `EC2_USER`
